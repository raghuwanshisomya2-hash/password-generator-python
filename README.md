# Password Generator (Python)

This project is a simple Python-based password generator that helps users create strong and secure passwords.

## Features
- Custom password length
- Option to include uppercase letters
- Option to include lowercase letters
- Option to include numbers
- Option to include special characters

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run `password_generator.py`
3. Follow on-screen instructions

## Security Note
Generated passwords are random and suitable for general use. For production-level security, cryptographic libraries are recommended.
